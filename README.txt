SupermotoLK – Simple Ecommerce Site
-----------------------------------

This website loads product data from Google Sheets.

Google Sheet Columns:
id | title | category | price | image | description | in_stock

To enable:
1. Open script.js
2. Replace YOUR_SHEET_ID with your real Google Sheet ID.
3. Publish Google Sheet → "Anyone with link can view"
4. Upload this folder to Netlify or Vercel for free hosting.
